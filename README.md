## 🖥️ 개요
> Spring AOP를 활용하여 공통 기능(로깅, 예외 처리 등)을 모듈화한 학습용 프로젝트입니다.
> 
> 비즈니스 로직과 공통 기능의 분리
> 
> AOP 기반 로깅 및 예외처리 구현 학습
